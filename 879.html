三分钟学会搭建我的世界基岩版服务器

Jun 24, 2019 |
折腾

目录

  2. 2. 连接服务器
  3. 3. 安装依赖
  4. 4. 上传服务端
  5. 5. 启动服务器

最近博主突然想要重新体验一下我的世界的乐趣，决定自己动手搭建一个服务器。初尝试是搭建 Java 版，花了半个小时研究，但在登陆测试时发现账号因长时间未用已被 Mojang 删除。不过幸运的是，博主还有基岩版账号，于是开始折腾基岩版服务器的搭建。

需要的软件

  1. Xshell - 用于远程操作服务器 官网链接
  2. WinSCP - 文件传输软件 官网链接
  3. Ubuntu 18.04 服务器 - 作为服务器操作系统
  4. 基岩版服务器端软件 - 从Minecraft官网下载 下载链接

连接服务器

首先，使用 WinSCP 连接到你的服务器。

成功连接后，右键新建一个名为 mc 的文件夹用于存放服务器文件。

安装依赖

接下来，打开 Xshell 并连接到你的服务器，执行以下命令来安装必要的软件：

apt install unzip
apt-get -t buster install libc6
apt-get install libstdc++6
apt install screen

上传服务端

前往 Minecraft 官网下载最新的基岩版服务器端压缩包，并通过 WinSCP 上传至 mc 目录。

上传完成后，在 Xshell 中输入 cd /root/mc 并解压文件：

unzip 你的压缩包文件名.zip

解压完成后，你将看到类似以下的文件结构：

启动服务器

为了防止断开 Xshell 与 服务器 的连接后会服务端关闭，先进入 screen 后再进入项目目录并启动服务器：

sudo screen
cd /root/mc
LD_LIBRARY_PATH=. ./bedrock_server

至此，大功告成！我的世界基岩版服务器已经顺利启动！
