三分钟学会搭建我的世界基岩版服务器

2019年06月23日 400 字 大概 1 分钟

最近突然想玩我的世界，想着搭建一个我的世界服务器玩玩，一开始搭建的是java版，研究个半个小时，最后打算上号测试是否成功的时候，突然发现我的账号由于长时间(半年多)没登录被Mojang删号不过幸好咱还有基岩版账号 ，所以去折腾基岩版服务器。

需要的软件

  1. Xshell（https://www.netsarang.com/zh/xshell/）
  2. WinSCP（https://winscp.net/eng/docs/lang:chs）
  3. 服务器 Ubuntu 18.0.4
  4. 服务器端软件（https://www.minecraft.net/zh-hans/download/server/bedrock/）

连接服务器

打开WinScp连接到我们的服务器

连接成功后右击->新建->文件(这里以mc为例)

安装依赖

打开Xshell连接服务器 输入以下四个命令安装我们需要的软件

apt install unzip 
apt-get -t buster install libc6
apt-get install libstdc++6 
apt install screen

上传服务端

前往 https://www.minecraft.net/zh-hans/download/server/bedrock/ 下载MC官方服务器端软件，下载完成后直接拖入WinScp右侧上传到mc目录。

上传后在xshell输入cd /root/mc命令定位到mc目录输入unzip 你下载的压缩包文件名（例如：unzip bedrock-server-1.11.4.2.zip ） 解压完成后如图

运行服务器

一切正常的话即可开始启动服务器 由于断开Xshell与服务器的连接,MC服务端也会被杀掉,所以我们需要先输入 sudo screen

然后在进入项目目录cd /root/mc 最后使用以下命令启动服务器LD_LIBRARY_PATH=. ./bedrock_server到此大功告成
