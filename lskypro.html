Lsky Pro(兰空图床)—又一款单纯的图床程序

2019年03月11日 620 字 大概 2 分钟

演示截图

（预览地址：https://pic.5ime.cn


主要特性

  1. 支持第三方云储存，支持本地、阿里云OSS、腾讯云COS、七牛云、又拍云。
  2. 支持多图上传、拖拽上传、上传预览、全屏预览、页面响应式布局。
  3. 简洁的图片管理功能，支持鼠标右键、单选多选等操作。
  4. 强大的图片预览功能，支持响应式。
  5. 支持全局配置用户初始剩余储存空间、支持单个设置用户剩余储存空间。
  6. 支持一键复制图片外链、二维码扫描链接。
  7. 支持设置上传文件、文件夹路径命名规则。

安装需求

  • PHP版本 ≥ 5.6(建议使用PHP7+)
  • mysqli支持
  • fileinfo拓展
  • curl拓展
  • zip拓展
  • rewrite

安装教程

  1. 下载兰空，上传至web运行环境，解压。
  2. 设置运行目录为public。
  3. 配置Rewrite规则：

Nginx：

location / {
    if (!-e $request_filename) {
        rewrite ^(.*)$ /index.php?s=$1 last; break;
    }
}

Apache: Apache直接使用.htaccess即可
4、访问首页，未安装自动跳转至安装页面，根据页面提示安装即可。
5、安装完成以后请设置runtime目录0755权限，如果你使用本地存储，public目录也需要设置为0755权限
修改网站运行目录 默认程序的入口文件在 public目录下，所以需要把public目录设置为网站运行目录，入口文件位置的设计是为了让应用部署更安全，public 目录为web可访问目录，其他的文件都可以放到非WEB访问目录下面。
而你如果安装时没有设置，使用 http://域名/public的方式来访问站点，会导致css和js等静态资源无法获取。
如果你 无法 或 不会 设置运行目录，可以将public目录下的所有文件和文件夹（包括.htaccess文件）移动到根目录 (和 application 文件夹同级)即可，尽管我们不推荐你这么做，这样会导致应用程序核心文件暴露在外。
移动文件以后，打开根目录index.php文件，修改如下：

<?php

// [ 应用入口文件 ]
namespace think;

// 根目录常量配置(和本文件同级)
define('ROOT_PATH', __DIR__ . DIRECTORY_SEPARATOR);

// 加载基础文件
require __DIR__ . '/thinkphp/base.php';

 // 执行应用并响应
Container::get('app')->bind('index')->run()->send();

注意：请不要使用记事本打开修改，修改完成后保存即可。
