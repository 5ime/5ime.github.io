GNU GRUB 是一个来自 GNU 项目的多操作系统启动程序。它允许用户可以在计算机内同时拥有多个操作系统，并在计算机启动时选择希望运行的操作系统。

最近安装了 Windows+Ubuntu 双系统，感觉美滋滋，但是开机时会出现十分简（丑）洁（陋）的 GRUB。对于使用 Linux 作为个人操作系统的用户来说，这也太丑了吧

所以，一起来美化自己的 Linux GRUB 引导界面吧

下载主题

前往 Gnome-Look 下载主题（国外网站，访问过慢请自觉挂梯子）。点击右侧侧边栏 GRUB Themes，就会出现几百个 GRUB 主题

有选择困难症？滑到底部，右侧侧边栏有 Popularity 最受欢迎排行榜

点击右上角 Download，弹出下载选项，大家可以根据自己的硬件和喜好选择下载不同尺寸。这里我选择下载文件 Tela-1080p.tar.xz

安装主题

主题下载完成后，进入下载目录。我的下载目录是 Download，具体情况请自行调节。然后解压文件，运行 install.sh 进行自动安装

cd /download # 进入下载目录
tar -Jxf Tela-1080p.tar.xz # 解压主题
cd Tela-1080p # 进入主题目录
sudo ./install.sh # 运行自动安装脚本
sudo update-grub # 更新配置文件
reboot # 重启电脑

安装完成
