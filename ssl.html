Dec 8, 2017 |

|
SSL

最近朋（ji）友他们全都换上了 HTTPS，我饥渴难耐按耐不住想上（zhuang）HTTPS（bi）的心情，试了一下居然可以了

于是就果断上了 HTTPS 证书，这个看似高大上的SSL泛域名证书实际上只要1块，超划算来着~

首先把的所有图片的链接换成支持 HTTPS 的，多亏了 WordPress更换域名插件 的帮助，

通过插件可以很方便地批量替换掉，那么多东西手动换的话不知道要换到什么时候了==

当然你也可以通过数据库批量替换

UPDATE wp_posts SET post_content = replace(post_content,'旧域名','新域名');
