沙拉查词自用CSS美化样式

2021年08月05日 170 字 大概 1 分钟

应该是我目前用过的最好用的网页划词翻译吧，毕竟支持自定义CSS样式。
目前在Gtihub Star 7.7K https://github.com/crimx/ext-saladict
按照自己的感觉改了改，应该能看出哪个是改过的吧2333

点击查看大图

/* 沙拉查词设置-查词面板-自定义查词面板样式 */

.menuBar {
	padding: 10px
}

.dictItemHead {
	padding: 6px;
	height: auto;
	margin: 4px 10px;
	background: #f5f5f6;
	border-radius: 4px
}

.dictItem-Body {
	padding: 0 20px
}

.saladict-theme {
	--color-brand: #fff
}

.dictItemHead-Logo {
	display: none
}

.menuBar-SearchBox {
	color: #666;
	background-color: #f5f5f6;
	border-radius: 4px
}

.menuBar-Btn_Icon,.menuBar-Btn_Icon-fav,.menuBar-Btn_Icon-history {
	fill: #000
}

.mtaBox-DrawerBtn,.waveformBox-DrawerBtn {
	height: 20px
}

textarea {
	border: 0;
	outline: none;
}
