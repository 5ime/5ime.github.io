TenAPI issue bankruptcy

Jul 23, 2024 |
文章
|
API · 项目 · 爬虫 · 去水印
停跑运路公了告!?

TenAPI 自上线起的六年间一直致力于为用户提供免费、高效和可靠的 API 服务。但 TenAPI 的现有架构和功能已经难以满足未来的需求。即便再次进行重构和优化，依然面临很多技术瓶颈和限制。博主在多次权衡之后，决定即日起停止维护，同时 2024 年 12 月 24 日正式下线 TenAPI。

事实上，早在 I Am I 年度简报 - 2023 中就已经提到即将下线的计划。

今年一开年博主便对 TenAPI 进行了彻底的重构，详见 TenAPI Update 2.0。同时，博主使用 Github Actions 实现了文档的自动部署，仓库地址：TenAPI 食用文档。通过文档的 更新页面 可以看到，今年上半年博主一直在频繁地进行维护。但随着时间的推移，维护工作逐渐减少，博主也逐渐感到心力交瘁。

不少数据源站已经宣布了关闭或转移的计划。例如：音悦台、轻视频、Vue Vlog、小咖秀、巴塞视频、before避风等平台相继关闭。这些平台的关闭导致 TenAPI 的数据源逐渐减少，使得服务的稳定性和可用性受到严重影响。此外，抖音、快手、蓝奏云等平台的反爬规则更新频繁且越来越强大，使得博主在获取数据时遇到了越来越多的困难，极大地增加了维护成本和技术难度。愈发感觉 TenAPI 要走到生命的尽头了。

其次，在 2021 年的 关于最近 一文中提到，由于调用量激增，TenAPI 需要改版，但最终仅仅是接入了 Cloudflare 并启用了防火墙以达到限流目的，详见 关于 Ten·API 防火墙的配置。同时，服务器也从腾讯云、Azure 过渡到了甲骨文以降低成本。

由于无法负担流量和服务器费用，所以 API 将进行大改版。未来，调用API将需要注册账号，且每人每日将有调用次数限制。

随着用户的增多，各种问题也接踵而至。六年来一个人维护确实心力交瘁。目前来看，TenAPI 最大的问题就是未做本地化存储。例如，网易云热评、抖音去水印等接口均可以将某些唯一的字段存储到数据库中，以避免重复请求。

在此，特别感谢六年间一路支持 TenAPI 的用户和开发者。你们的反馈和建议是 TenAPI 不断改进和进步的重要动力。虽然 TenAPI 将要下线，但博主相信在未来的某个时刻，会有更优秀的服务和产品出现，继续为大家提供便利。

再次感谢大家的支持与理解。希望未来大家还能在其他领域和项目中再会。
