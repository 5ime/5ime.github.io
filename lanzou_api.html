Aug 7, 2020 |
技术

PHP · API

使用方法

  • url:蓝奏云外链链接
  • type:直接下载

支持链接

  • *.lanzous.com
  • *.lanzoui.com
  • *.lanzoux.com

请求示例

  • 无密码 https://tenapi.cn/v2/lanzou/?url=https://www.lanzous.com/i8fclgh
  • 有密码 https://tenapi.cn/v2/lanzou/?url=https://www.lanzous.com/itahfehy1bc&pwd=d17u
  • 直接下载 https://tenapi.cn/v2/lanzou/?url=https://www.lanzous.com/itahfehy1bc&pwd=d17u&type=down
