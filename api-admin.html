食用方法

Github地址：https://github.com/5ime/api-admin

此程序基于 ThinkPHP5.1，支持的 PHP 版本为 7.0 至 7.3 之间

Nginx 配置

对于使用 Nginx 的用户，请设置如下伪静态：

location / {
    if (!-e $request_filename){
        rewrite  ^(.*)$  /index.php?s=$1  last;   break;
    }
}

Apache 用户无需特别配置，程序运行目录设置为默认即可。

安装程序

将程序上传至网站根目录，通过访问 域名/install 来启动安装程序。

默认账号和密码分别为 admin 和 123456。如需添加 API，请将 API 文件放入 api 目录。

文档编写及格式规范请参考：TenApi文档规范
