自 2017 年起，我每年都会写一篇年度总结，但多数时候不过是些流水账，平淡无奇，毫无意义。

到了 2022 年，我开始尝试记录过去一年中的所见、所闻，以及所思、所想。然而如今看来，回忆总容易触及痛点，令人不免陷入情绪的低谷。于是，特意挑了一天心情尚可的时候动笔，希望文字能更坦然些。

WebSite

八月时，百度统计提示站点违规，且相关数据已被屏蔽。但经排查，貌似原因如下（自家统计产品做不好过滤，与我何干？）。注销百度统计后，网站迁移到谷歌统计。

但如今看来，统计的意义早已模糊。对 SEO 的记忆还停留在 清风算法 和 哪吒算法 时代。

Blog

这一年，网站更新的频率跌至历史最低点。年中时，便在 写在当下 一文中提到过部分原因。当下的自己，似乎已失去了产出新内容的能力。正如 2018 年在个人介绍中写道：

太简单的文章发了水，太难的文章写不出，中间的文章都有人写过。

打开桌面名为 烂尾文章 的文件夹，看到里面沉睡的文字，灵感似乎已一去不返。那些未完待续的篇章，在无声地等待，但我已无法给予它们生命。

API

年中逐渐觉得维护 API 站已无太大价值，遂发布了 TenAPI Issue Bankruptcy 一文。

原计划在复活节当天下线（写到这我才意识到把平安夜当成复活节了😅），未料服务器日志爆满，磁盘直接撑爆，随即在 2024-11-27 13:04 停运。

把近 30G 的网站日志拖下来分析了一番（三月至十一月），数据是冰冷的，但它似乎在诉说什么。

Lab

整个 I Am I の 实验室 本质上就是 TenAPI 接口的调用示例，随着 API 的停运，实验室也随之失去了意义。

曾计划重构这个项目，然而当我重新克隆并查看代码时，发现它就是一坨💩，理解不了以前为何自己这样写，遂作罢。

不过后期准备用 Vue3 和 Vite 进行彻底重构。同时写一些真正有趣的项目，使之真正的符合 实挖验坑室 的名字。

Coding

回顾上半年每天下班都会抽出时间来写代码，而现在每天回家后只是单纯的发呆，桌面上的电脑已然积满灰尘，内心也空空如也。而去年展望中所提到的 写更多的代码吧 似乎也未曾实现…

以前，我总觉得写代码是可以边写边学的，如今才发现，这种想法有些本末倒置了。

由此导致的基础不牢，每当在抖音刷到其他人写代码时，可以清晰地感受到的差距。自己投入的精力似乎还是不够，代码的质量和效率都远远不足。

与此同时，对代码的完美主义追求，导致今年反复陷入代码重构的怪圈。

今年用 daisy UI 和 Tailwind CSS 重构了 MoeCTF，但某人一句 黑白灰好丑 使得我心碎。

不过，在重构过程中，对前端的理解有了新的认识，也发现了许多以前未曾注意到的细节。整个项目算是已经完工，今年也支撑了一场 100 余人的网络安全赛事。

而去年所提到的 Loki-Labs 已然成为烂尾工程，原因之一是部署麻烦，且许多漏洞有版本限制。当前计划是将其与 Notes 项目联动（知识库 + 配套题目），通过重构 VitePress 来实现动态靶机的分发，减少用户对环境搭建的困扰。

Contest

去年，几乎每场比赛我都参加了，大小赛事不曾错过。然而，今年失去了学生身份，且已获评省技术能手，种种限制使得我只能参与少数几场比赛，且几乎都是二等奖😂（大金砖、省赛、强网杯、深信服云对抗…）

于是，今年更多的精力投入到了竞赛命题领域，例如 NKCTF 2024、某国赛和省赛的电子取证、数据安全赛题的设计等。

今年，与学长以 天命 为名参加了网鼎杯半决赛。整个过程中读不懂题和眼神不佳导致处处出问题。自强网杯以来便察觉到自己的阅读障碍，时常无法准确理解题目想表达的含义。

与朋友聊起，才发现其实很多人同样踩坑。或许，这场迷局的锅，不在选手，而在出题人？（个个都想当谜语人😊）

渗透阶段的解法，与我设想的思路几近一致，但为何当时没打进去呢？或许当时因 开发者视角 看问题，反而陷入了过度复杂的思维怪圈。

尽管如此，也拿下了其他阶段部分题目的前三血，名次虽在预料之中，但终究感觉不尽如人意。或许以后更适合当吉祥物（只挂名，不添乱😋）

时常反思自己的能力，感觉自己依然差强人意。大概是我还不够努力，太过怠惰吧…

不过，正如队名所示：尽人事，听天命。

题外话

今年赴某地参加了一场非常高规模的内部比武，拿了两个模块第一，一个模块第二。然而，赛事期间我与某部门的对接工作却陷入了极端的混乱之中，差旅费拖了六个月才报销，而精神与物质奖励至今下落不明…

虽然事是好事，但人不是好人，权当外出旅游了🤣

Wait…

今年，在各大平台上一个漏洞都没挖，仅参加了几场省部级的攻防演练，且水了个 CVE 和 CNVD 证书。

年中时，朋友提起了 CERT-EU Hall of Fame，简单看了一下收录规则，随手提交了漏洞，没想到顺利入选。

不得不说，欧盟 CERT 响应速度真的快，从提交漏洞到上榜全程不到 7 小时，但时隔半年漏洞依旧未修补。

顺势了解了一些其他组织的规则，也成功登上了 UN Information Security Hall of Fame。

与欧盟相比，联合国 CERT 极其负责，审核周期在 90 天左右，漏洞修复后才会更新名人堂。

尽管这些名号是一些华而不实的东西，但未来或许可以将它们挂在博客个人简介里，写成这样的戏谑🤣

非对立

今年，说过最多的词应该是 戒酒，但好友对我的评价是：口嫌体正直。不过和往年相比，今年喝酒确实少了很多，可这谁又能说得清呢？

今年上半年，由于个人档案出了些许问题，需提供毕业证书。经过一番周折，联系上了以前学校的主任，经过一番亲切又友好的交流后，不仅收到了阔别五年的毕业证，还成为了所谓的优秀毕业生。

遥想当年发了一篇批判性的文章 Uptime >>16 years，一阵复杂的情绪涌上心头。当年有多痛苦，如今就有多不适。讽刺的是，时间似乎在不断地验证世界的荒诞。

而整个下半年像陷入轮回，一切似曾相识，终点早已预见，而过程却难以摆脱。目前在这座城市，已经生活了四年，但始终感受不到所谓的归属感。像一名永远的局外人，默默地观察，却始终融不进去。

近一个月反复做梦，时常梦境交织，仿佛在现实与虚幻之间游走，从而导致记忆混乱。虽然这一切无所谓，毕竟终究都会被遗忘。

好在目前已有所改善，已能分清梦与现实，只是睡眠质量略显下降，且时常伴随睡眠瘫痪。

深秋，是我最喜欢的季节。万物萧瑟，树叶飘零，整个世界似乎都在回归一种沉寂，这份萧条感与我本身不谋而合。而眼前的安静，可以带来一种微妙的慰藉。

然而，奇怪的是，同事们却时不时地对我表达关心，总是无端地安慰我。朋友聊天时也会随口来一句：“感觉你好孤独，文字中透露出一种凄凉。”他们的语气中，总透着一种惋惜与怜悯。

但我觉得自己一切都挺好，这是我始终无法理解的事情。孤独，真的像他们说的那样糟糕吗？或许，他们从未理解过我所追求的安静与自由。

今年结识了许多新朋友，大多数都是值得交往的好人（只有一个不算是好人？）。至于好与坏的界限，每个人心中都有自己的答案，也许无关对错，只是视角不同罢了。

相比往年，今年外出的次数明显增多（非常感激大家愿意带我玩，尽管每次都需要做足心理准备）。过去那种与周遭环境的隔阂感，似乎也在慢慢被冲淡。街道的喧嚣、陌生的对话，如今都不再让我感到那么疏离。

迈出门槛，确实需要一份勇气，但每一次尝试，似乎都让我离那个更坦然的自己近了一步。

嗯，生活属于自己，快乐才最重要。

最近，我下载了慢脚，翻看评论时，记忆不由得翻涌而出。那些年似乎还停留在原地，未曾离开。仔细想来，这种执念又有什么意义？纠结于过往，终究无法改变什么。

这一年里，同事们热衷于为我介绍对象（好在由于找不到和我年龄相仿的人，最终未果），我始终觉得一个人生活也挺好。没有牵绊，安静自在，自成一隅。

朋友们倒是常常带着一些荒唐的传闻来找我求证，想从我口中得出真相。而我，早已对这些传闻心如止水，传来传去，面目全非的故事已然失去了意义。

2025/12/31 晚更新，今晚吃饭时终于听到了传闻的初代版本，才发现从源头开始，一切就已经充满了荒谬与不解。为何要回去上课？为何会选择玩那个游戏？一切似乎都有些说不清的奇怪原因，通过不同人的荒诞演绎，这一切变得愈发扑朔迷离~~（似乎他们每个人都很了解我？）~~。

未来展望

以前，总想要寻找所谓的意义，却始终不得其解。如今，或许已经找到了答案：人生没有意义，只有过程。

希望以后真正可以做到：

宠辱不惊，闲看庭前花开花落；去留无意，漫随天外云卷云舒。
