一眨眼API运行快五年了，Ten·Api正式发布-免费提供Api服务，近期对TenAPI进行了一次较大的维护…

API迈入了第二个版本，文档迈入了第四个版本，接口数量移除了大半(现存接口均可正常使用)。

接口地址

博主换了一种更方便维护的方式来写接口，所以接口地址也发生了变化，原接口地址将于2023年4月1日停止服务。

原地址：https://tenapi.cn/api
新地址：https://tenapi.cn/v2/api

返回格式

所有接口除部分头像获取类会直接跳转外，其他接口均返回json格式数据， 同时固定返回 code 、msg 、data 三个字段，其中 code 为状态码，msg 为状态信息，data 为返回的数据

{
  "code": 200,
  "msg": "success",
  "data": {
    // 返回的数据
  }
}

新增

用户IP地址查询(支持IPv4和IPv6)，今日头条热点文章，今日头条热搜榜

移除

QQ上传透明头像，网易云歌单信息获取，网易云/QQ歌曲直链获取，微信短网址生成，QQ短网址生成，新浪短网址生成，QQ短网址拦截查询，网站ICP备案查询，网站Whois查询，网站标题获取，ico网站图标获取，网站收录量查询，网站权重获取，同IP网站查询，服务器信息查询，垃圾分类查询，申通快递查询

图集和视频移除了ins的解析，已被ins封禁两个账号，暂时无解，同时移除音悦台, 轻视频, Vue Vlog, 小咖秀, 巴塞视频，before避风，均已倒闭

合并

Bilibili关注/粉丝数获取与哔哩哔哩用户信息合并，Bilibili视频封面获取与哔哩哔哩视频信息合并，删除了所有的单视频去水印解析，全部合并

其中两个聚合接口，一个是短视频解析，一个是短视频图集解析，支持的平台如下

短视频

(快手和西瓜视频不太稳定，其他平台应该没什么大问题) 皮皮虾, 抖音, 微视, 快手, 6间房, 哔哩哔哩, 微博, 绿洲, 度小视, 开眼, 陌陌, 皮皮搞笑, 全民k歌, 逗拍, 虎牙, 新片场, 哔哩哔哩, Acfun, 美拍

图集

抖音，快手，皮皮虾，最右

文档

文档中的接口地址已经全部修改为新地址，同时部署方式改为了 Github Action，由于重写了文档，所以文档内的链接有所变动。

旧文档地址：https://docs.tenapi.cn/old
