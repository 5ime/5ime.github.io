鉴于某些原因，故记录此篇文章，以作参考。

浏览器打开 Kali VMware版 下载页面，选择合适的版本进行下载，这里博主下载的 Kali Linux VMware 64-Bit(7z)

下载完成后将压缩包放到合适的位置然后解压文件夹 双击 Kali-Linux-2020.3-vmware-amd64.vmx 启动虚拟机。

选择 我已复制此虚拟机 即可正常进入Kali系统。默认账号密码都是 kali

更换镜像源

点击上方托盘上的 Terminal Emulator 启动命令行 输入以下命令打开镜像源配置文件(粘贴的快捷键是 Ctrl+Shift+V )

sudo vi /etc/apt/sources.list

命令行会提示你输入 kali(当前用户) 的密码，这里输入 kali 即可(注意Linux输入密码的时候没有任何显示 输入完直接回车即可)

进入 Vim 编辑器后,输入法改为英文输入，按 a 键开启编辑模式 此时左下角会显示-- INSERT -- 表示已经入可编辑模式

使用#注释掉原本的镜像源,更换为国内的镜像源，这里博主用的阿里云提供的镜像源。

更换镜像源后，按ESC键退出编辑模式 此时左下角-- INSERT --提示消失。此时输入:wq强制保存退出

#中科大
deb http://mirrors.ustc.edu.cn/kali kali-rolling main non-free contrib
deb-src http://mirrors.ustc.edu.cn/kali kali-rolling main non-free contrib
 
#阿里云
deb http://mirrors.aliyun.com/kali kali-rolling main non-free contrib
deb-src http://mirrors.aliyun.com/kali kali-rolling main non-free contrib
 
#清华大学
deb http://mirrors.tuna.tsinghua.edu.cn/kali kali-rolling main contrib non-free
deb-src https://mirrors.tuna.tsinghua.edu.cn/kali kali-rolling main contrib non-free

更新软件

在命令行内输入以下内容进行漫长更新。遇到提示直接输入 y 即可

sudo apt-get update && apt-get upgrade && apt-get clean

更换编码

更新完成后输入下面的内容 选择编码 在弹出的框中用 空格键 选中 ZH_CN.GBK GBK 和 ZH_CN.UTF-8 UTF-8 之后，按下 回车键 确定进入下一步。接着选择 ZH_CN.UTF-8 UTF-8 然后按 回车键 确定

sudo dpkg-reconfigure locales

在命令行内输入以下命令安装中文字体,安装完成后使用 sudo reboot 重启系统，到此完成

sudo apt-get install ttf-wqy-microhei ttf-wqy-zenhei xfonts-wqy

更换完成

这里建议保留旧的名称，因为输入中文确实不方便，不要选择更换名称
