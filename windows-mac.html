玩转Windows之Mac风格

2018年05月22日 423 字 大概 1 分钟

效果图

本次教程使用的系统为Windows 10其他请自测。 先上几张效果图

更改前

更换后



所需工具

  • Dock
  • Launchpad
  • Finder

隐藏任务栏

（如果你觉得Dock和任务栏共存不影响美观你可以忽略此步骤） 右击任务栏任务栏设置-个性化``-任务栏-然后开启在桌面模式下自动隐藏任务栏

字体

你可以选择Mactype它能接管 Windows 的 GDI 字体渲染功能，实现比 Mac OS 更华丽的字体渲染效果！ 或者下载一些Mac特有的一些字体，把打包的所有字体文件安装下即可。点我下载 emmm，安装字体我就不用教了吧。

Dock

Mac OS 下原生 Dock 效果图

Windows 下模拟 Dock 效果图

点我下载MyDock,下载解压MyDock.zip后运行Dock.exe即可运行本程序 右键 Dock 分割线上方的空白处，点击进入Dock 偏好设置。设置可以参考我的。



其他选项均为默认，重启软件生效配置。

Launchpad

Mac OS 下原生 Launchpad 效果图

Windows 下模拟 Launchpad 效果图


上面安装的MyDock软件集合了Launchpad功能，launchpad目录下放入快捷方式，MyDock会自动识别将其加入Launchpad中。

FinderFinder

Mac OS 下原生 Finder 效果图

Windows 下模拟 Finder 效果图


点我下载MyFinde，解压MyFinder.zip文件，然后双击运行MyFinder.exe即可打开 Finder 点击左上角的Finder-偏好设置 进入软件的设置界面。可以参考我的设置。

其他选项均为默认，重启软件生效配置。 到此大功告成。
