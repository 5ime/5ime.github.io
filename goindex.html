Goindex 将 Google Drive 打造成网盘

2019年09月05日 折腾 约 213 字 大概 1 分钟

功能

  1. 部署在 CloudFlare Workers 的小程序。
  2. 可以将 Google Drive 文件以目录形式列出，并直连下载。
  3. 流量走 CloudFlare，网速由 CloudFlare 决定。

演示地址

  • 访问演示: Goindex Demo
  • 查看源代码: Goindex on GitHub

部署教程

获取Auth认证码

  1. 访问 Auth Code Generator，登录你的 Google 账号以获取认证码。

  2. 将出现的 code 复制到Auth认证码输入框。

配置CloudFlare Workers

  1. 目录ID可空着，也可填写团队盘目录ID

  2. 点击生成代码，复制生成的代码。

  3. 登录到 CloudFlare Workers，点击 Get started with Workers 创建一个新项目，由于我刚才新建了，忘记截图了，所以没有图片。

  4. 新建完成后，点击Edit进入项目编辑页面。

  5. 将在https://install.gd.workers.dev/生成的代码粘贴到编辑器中并保存更改即可
