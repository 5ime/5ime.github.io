所需工具

  1. Fiddler：下载链接
  2. M3u8下载器：GitHub Release

配置 Fiddler

首先，需要配置 Fiddler 来捕获 HTTPS 流量：

  • 打开 Fiddler，点击菜单栏中的 Tools，选择 Options。
  • 安装 SSL 证书以允许 Fiddler 监听 HTTPS 通信。

中间人注入

在 Fiddler 中使用脚本注入：

  • 选择 Fiddler Script 标签页，点击 OnBeforeResponse，会自动定位到我们需要添加代码的地方。
  • 在适当的位置粘贴以下 JS 代码，然后点击 Save Script 保存脚本。
var sToInsert = "<script src='https://cdn.jsdelivr.net/gh/Tencent/vConsole@3.8.1/dist/vconsole.min.js'></script><script>var vConsole = new VConsole();</script>"
oSession.utilDecodeResponse();
oSession.utilReplaceOnceInResponse('</head>', sToInsert + '</head>', 0);

监听流量

打开钉钉回放页面，并激活 vConsole：

  • 在回放页面的右下角会看到一个新的 vConsole 按钮。
  • 单击此按钮，并执行以下代码以查看网络活动。
vConsole.showTab("network");

  • 流量中会显示多条记录，选择最高的那一条单击。

  • 得到回放视频链接 URL。

下载 M3U8

使用 N_m3u8DL-CLI 工具下载视频：

  • 打开下载器，粘贴复制的 M3U8 链接，开始下载。
